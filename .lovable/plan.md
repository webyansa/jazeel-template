

# خطة بناء منصة جزيل - واجهة فرص المنح

## ملخص المشروع
بناء واجهة احترافية حديثة لمنصة جزيل (منصة منح للقطاع غير الربحي) تتكون من صفحتين: صفحة عرض فرص المنح وصفحة تفاصيل الفرصة، مع تصميم RTL كامل وهوية بصرية مخصصة.

## الخطوات

### 1. إعداد الهوية البصرية والبنية الأساسية
- تحديث ملف `styles.css` بألوان جزيل (Navy #07133F, Green #1DBA8C, Mint #EAF8F3)
- إضافة خط **Tajawal** من Google Fonts
- ضبط الـ `__root.tsx` ليدعم RTL بشكل أصلي (`dir="rtl" lang="ar"`)
- نسخ شعار جزيل إلى المشروع

### 2. بناء المكونات المشتركة (Reusable Components)
- **Header**: هيدر ثابت مع شعار جزيل، روابط التنقل، زر تسجيل دخول، قائمة جوال
- **Footer**: فوتر احترافي مع نبذة عن جزيل، روابط، أيقونات تواصل
- **GrantCard**: بطاقة فرصة المنح مع شعار الجهة، العنوان، الوصف، المدة المتبقية، badges الحالة، أزرار الإجراء
- **Badge**: شارات مخصصة (مفتوحة، تغلق قريباً، جديد، إقبال مرتفع)
- **NewsletterCTA**: قسم الاشتراك في النشرة البريدية

### 3. صفحة فرص المنح (`/` - الصفحة الرئيسية)
- **Hero Section**: عنوان جذاب "استكشف فرص المنح المناسبة لمشاريعك" مع حقل بحث رئيسي ومؤشرات إحصائية (عدد الفرص، الجهات المانحة، المجالات)
- **شريط البحث والفلترة**: بحث بالعنوان، فلتر المجال، المنطقة، حالة التقديم، نوع الدعم، مع زر إعادة تعيين
- **عرض البطاقات**: Grid على الديسكتوب، stacked على الجوال، مع بيانات تجريبية واقعية بالعربية (8-10 فرص)
- **حالة عدم وجود نتائج**: تصميم أنيق مع اقتراح تعديل البحث
- **Pagination**: تصميم حديث للتنقل بين الصفحات
- **قسم النشرة البريدية**: CTA للاشتراك

### 4. صفحة تفاصيل فرصة المنح (`/grants/$grantId`)
- **Hero/Header Summary**: شعار الجهة، عنوان الفرصة، الحالة، العد التنازلي، أزرار (التقديم، إعداد المشروع، مشاركة، حفظ)
- **Layout ثنائي الأعمدة**: محتوى رئيسي + sidebar sticky
- **المحتوى الرئيسي**: أقسام (نبذة، أهداف، معايير القبول، شروط التقديم، المجالات المستهدفة، آلية التقديم بخطوات مرقمة، أسئلة شائعة بـ Accordion)
- **Sidebar**: معلومات سريعة (الجهة، الموعد، الحالة، المدة، النطاق، المجال، أزرار الإجراء)
- **فرص مشابهة**: 3-4 بطاقات في أسفل الصفحة
- **CTA نهائي**: "ابدأ إعداد مشروعك عبر جزيل"

### 5. تحسينات UX
- تصميم responsive كامل (جوال، تابلت، ديسكتوب)
- Hover/focus/active states أنيقة على جميع العناصر التفاعلية
- Loading skeletons للبطاقات
- Subtle transitions وanimations
- Accessibility جيدة (contrast، focus states، labels)
- White space ممتاز وتسلسل بصري واضح
